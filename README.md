Real-time Timeline Event inspector for myyogateacher.com

MYT Event Inspector is an internal Chrome extension for the MyYogaTeacher team to inspect Timeline Events firing on myyogateacher.com in real time.

Open the side panel while browsing any myyogateacher.com page to see every UI event as it fires — page views, CTA clicks, form submissions, and more — along with the full event payload including metadata, device context, UTM parameters, and user identity state.

Built for:
Marketing, growth, engineering and data teams to verify event tracking, debug missing events, and QA new implementations without opening the Network tab.

Key features:
1. Real-time event stream accumulated across page navigations
2. Event type badges (View, Click, Capture, Success, Failure)
3. Expandable metadata, device, and session sections per event
4. Page change separators showing navigation flow
5. Session header showing anonymous vs logged-in state
6. Filters out non-Timeline Events automatically

Internal use only. Not intended for public distribution.
